# Healing Song Mood Player

현재 기분에 따라 추천 노래를 고르고, 유튜브 내장 플레이어 또는 새 창 링크로 바로 감상할 수 있는 정적 웹페이지입니다. 몽환적인, 활기찬, 애틋한, 가슴이 벅차오르는 분위기의 곡을 한 화면에서 탐색할 수 있도록 구성했습니다.

## 사용 방법

1. 상단 메뉴에서 `추천곡`으로 이동합니다.
2. 현재 기분에 맞는 필터를 선택합니다.
3. 노래 카드 안의 유튜브 플레이어로 바로 재생하거나, `유튜브에서 바로 듣기` 버튼을 눌러 새 창에서 감상합니다.
4. 직접 추가하고 싶은 곡이 있으면 `노래 추가` 영역에 제목, 가수, 유튜브 링크, 기분을 입력합니다.
5. 추가한 곡은 브라우저의 localStorage에 저장되어 새로고침 후에도 유지됩니다.
6. 필요하면 상단의 `다크 모드` 버튼으로 화면 분위기를 바꿀 수 있습니다.

## 주요 기능

- 기분별 추천곡 필터링
- 유튜브 링크 기반 내장 플레이어 표시
- 유튜브 원본 링크 새 창 열기
- 사용자가 직접 추천곡 추가
- 추가한 곡 localStorage 저장
- 다크 모드 / 라이트 모드 전환
- 추천 문장 자동 전환, 멈춤, 다음 추천 기능
- 데스크톱, 태블릿, 모바일 화면에 맞춘 반응형 카드 레이아웃
- 키보드 포커스 표시와 이미지 대체 텍스트를 포함한 접근성 보완

## 로컬 실행 방법

이 웹페이지는 `data/songs.json`을 불러오기 때문에, 파일을 직접 더블클릭하기보다 로컬 서버로 실행하는 방식이 안정적입니다.

### Windows 더블클릭 실행

1. ZIP 파일의 압축을 풉니다.
2. 폴더 안의 `start-local.bat`을 더블클릭합니다.
3. 브라우저에서 `http://localhost:8000`이 열리면 정상 실행된 것입니다.

### 터미널 실행

```bash
python -m http.server 8000
```

브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:8000
```
